Escalando para 1000 ou para 0: um jeito de pensar serveless para resolução de problemas reais

Nesta palestra conheça como passamos a resolver problemas reais do cotidiano da rotina do EasyCrédito, o maior Hub de Crédito do Brasil, com um jeito de pensar cloud native serveless. Para isso, descrevo como estamos usando recursos como o Cloud Run, Cloud Functions, Load Balancers e Cloud Armor em um modelo de escala horizontal e inteligente. Tudo isso em uma estrutura controlado por processos automatizados de deploy e monitoramento.
